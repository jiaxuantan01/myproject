# Member Management System

A simple backend-focused member management system built with Laravel and MySQL.

## Features

- User authentication
- Member management CRUD operations
- RESTful API implementation
- Basic role and profile management
- Responsive frontend interface

## Tech Stack

- Laravel
- PHP
- MySQL
- Vue.js / Blade
- HTML, CSS, JavaScript
- Nginx
- Linux

## Project Purpose

This project was built to strengthen practical backend development skills, including API development, database design, authentication flow, and MVC architecture implementation.
